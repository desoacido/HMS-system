# Hospital Management System (HMS)

 📌 Overview

This system is a Hospital Management System designed for barangay health centers. It manages patient records, referrals, and includes a machine learning feature for prediction.

 🛠️ Technologies Used

* PHP (Frontend and system logic)
* PostgreSQL (Database)
* Python (Machine Learning - Supervised Learning)

🤖 Machine Learning

The system uses a trained ML model based on a dataset (CSV) to predict possible conditions based on patient inputs.

⚠️ Note

* The system is fully functional in a local environment (XAMPP + PostgreSQL + Python).
* Machine Learning integration runs via Python scripts.
* Online deployment is still in progress.
